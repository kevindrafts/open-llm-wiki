## Start with this 15-minute Claude Code test:

**Step 1:** Open Claude Code in any folder

**Step 2:** Type: `/mcp`

**Step 3:** If you don’t have Perplexity MCP installed, ask Claude Code: **“Help me set up the Perplexity MCP.”** (all you have to do is paste the perplexity API key)

**Step 4:** Once installed, run this prompt:

```
Context

We are building/ running: [TYPE OF BUSINESS / PROJECT]

Targeting: [AUDIENCE + SPECIFICS]

What makes us different: [YOUR EDGE]

Current state: [WHERE YOU ARE NOW]

Task

Use Perplexity to deeply research [YOUR SPACE].

I want you to produce a clear, structured research brief covering:

Market landscape

Key competitors / service providers today

What’s working for them (channels, messaging, offers)

Customer pain points + gaps in the market

Pricing models (if relevant)

Output

Save everything as research.md in this folder.
```

**Step 5:** Watch it work.

## What you should see

Claude Code will:  
  
\- Fire up Perplexity  
\- Run multiple deep research queries  
\- Synthesize the findings  
\- Save your research in an organized markdown file

All without you leaving the terminal.

## Why this matters

Because the research is the prompt.

When Claude has real context, everything gets easier:

- Your marketing copy and assets get sharper
- Positioning stops sounding generic
- Content ideas stop looking like everyone else’s

Same model. Same you.

Different input → different output.

“Cast a WIDE net. Gather DEEP context. Let AI work. Outputs are just way better, when you have the right context, the right skills, and the right tools, prompt engineering is less important.”
